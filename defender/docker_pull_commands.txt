# Docker Pull Commands for MLSEC Defender Models

## Decision Tree Model
docker pull gamdhameet/gamdhameet-dt:latest

## Random Forest Ensemble Model
docker pull gamdhameet/gamdhameet-rf:latest

## XGBoost Model
docker pull gamdhameet/gamdhameet-xgb:latest

# To run the containers:

## Decision Tree
docker run -itp 8080:8080 --memory=1.5g --cpus=1 gamdhameet/gamdhameet-dt:latest

## Random Forest Ensemble
docker run -itp 8080:8080 --memory=1.5g --cpus=1 gamdhameet/gamdhameet-rf:latest

## XGBoost
docker run -itp 8080:8080 --memory=1.5g --cpus=1 gamdhameet/gamdhameet-xgb:latest
